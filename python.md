## 1.1  Python数学操作符
```
**  指数    2**3    8  
%   取余    22%8    6  
//  整除    22/8    2  
／  除法    22/8    2.75  
*   乘法    3*5     15  
-   减法    5-3     2  
+   加法    2+2     4   
``` 
以上操作符，优先级从高到低排列 

## 1.2  整型，浮点型和字符串数据类型
Python中最常见的数据类型：
* 整形（或int），表明值时整数
* 浮点型（或float）带有小数，如3.14
* 字符串，总是用单引号（‘）包围住字符串（如’hello‘）

*操作符作用于两个数字时表示乘法，作用域一个字符串和一个整型时表示字符串复制
例：
```python
str = 'abc'
println(str*5)//abcabcabcabcabc
```
*操作符不能作用域一个字符串和一个浮点型，因为你不能把一个任意字符串复制小数次
